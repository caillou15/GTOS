La commande fix/ permet de créer une variable.

Les variables principale à utiliser dans un fichier de commandes indexé sur Index.cpc

OS = { Nom de l'OS }
Version = { Version de l'OS }
Auteur = { Développeur(s) }
Org = { Entreprise/organisation}
Source = { Site internet }
contact = { Adresse E-mail }

Prog = { Cible du dossier de programmes }
Media = { Cible du dossier média }
Systeme = { Cible du dossier système }


ps : on remet toujours le répertoire "OS\" dans exe/ OS\OS.cpc car le répertoire courant est
TOUJOURS TOUJOURS : c:\Cpcdos\Systeme
donc préciser TOUJOURS le chemin d'accès complet!
